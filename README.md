#📍 IP Address Finder Web App

A simple and responsive web application that allows users to find details of any IP address, including location, timezone, and ISP (Internet Service Provider).

🚀 Features
🔍 Search any IP address
🌍 Displays geographic location (Country, City)
🕒 Shows timezone information
🌐 Displays ISP details
📱 Responsive design (works on mobile & desktop)
⚡ Fast API-based data fetching
🛠️ Tech Stack
HTML – Structure of the web app
CSS – Styling and layout
JavaScript – Logic and API handling

🔗 How It Works
User enters an IP address in the input field
Clicks on the search button
The app sends a request to an IP Geolocation API
The API returns:
Location
Timezone
ISP
The data is displayed on the screen
